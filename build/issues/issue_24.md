## Useful Code Snippets
```python
# run external files
%run myCode.py

# overwrite another file in another notebook
%writefile myCode.py

# display content of external file
%pycat myCode.py

#list all variables 
%who

# Share variables between notebooks
%store my data

#In another notebook
%store -r myData

# list all magic command 
%lsmagic
```
